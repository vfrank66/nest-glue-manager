## Description

A Nest library wrapper around aws-sdk to surface AWS Glue calls. 

## Installation

```bash
$ npm install nest-glue-manager
```

## Adding the Global Module

```node
GlueManagerModule.forRoot({
  region: 'us-east-1'
})

```
